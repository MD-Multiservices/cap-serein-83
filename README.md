# Cap Serein 83 - site Next.js

Site indépendant pour l'activité de conciergerie, prévu pour le sous-domaine :

`https://conciergerie.mdmultiservices83.fr`

## Installation

```bash
npm install
npm run dev
```

## Déploiement Vercel

1. Créer un dépôt GitHub séparé, par exemple `cap-serein-83`.
2. Importer le dépôt dans Vercel.
3. Ajouter le domaine `conciergerie.mdmultiservices83.fr` dans les réglages du projet.
4. Créer dans OVH l'enregistrement DNS demandé par Vercel.
5. Vérifier le certificat HTTPS.

## À faire avant publication commerciale

- Remplacer l'encadré provisoire « Médiation de la consommation » après signature avec un médiateur référencé.
- Vérifier les tarifs du partenaire ménage et linge.
- Vérifier le QR code.
- Vérifier que la RCP couvre la nouvelle activité.
